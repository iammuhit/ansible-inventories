## How to Build Inventory

```bash
## 
# Create a Virtual Machine
# 
# @see: https://developer.hashicorp.com/vagrant/docs/cli/init
# @see: https://developer.hashicorp.com/vagrant/docs/cli/up
# @see: https://developer.hashicorp.com/vagrant/docs/cli/ssh_config
# 
vagrant init -m ubuntu/jammy64
vagrant up --provider virtualbox
vagrant ssh-config

## 
# Verify SSH Connection
# 
# @see: https://www.ssh.com/academy/ssh/command
# @see: https://www.ssh.com/academy/ssh/config
# 
# Command Options:
# 
# -p <port>                         Port to connect to on the remote host
# -i <identity-file>                Identity key (private key)
# -o StrictHostKeyChecking=no       Disable host key checking
# -o UserKnownHostsFile=/dev/null   Don't store host keys
# 
ssh vagrant@ip-address -p 22 -i ./.vagrant/machines/default/virtualbox/private_key -o StrictHostKeyChecking=no

##
# Ansible Patterns & Pong Module
# 
# Try to connect to host
# Verify usable python and return pong on success
# 
# @see: https://docs.ansible.com/ansible/latest/inventory_guide/intro_patterns.html
# @see: https://docs.ansible.com/ansible/latest/inventory_guide/connection_details.html
# @see: https://docs.ansible.com/ansible/latest/collections/ansible/builtin/ping_module.html
# 
# ansible <pattern> -i <inventory-file> -m <module-name>
# 
ansible webservers -i inventory.yml -m ping
```
